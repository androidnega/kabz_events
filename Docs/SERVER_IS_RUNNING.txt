If you need to stop the current background server:

Option A - Find and Kill the Process:
  Get-Process -Name php | Where-Object {$_.MainWindowTitle -eq ""} | Stop-Process

Option B - Use Task Manager:
  1. Open Task Manager (Ctrl+Shift+Esc)
  2. Find "php.exe" process
  3. Right-click → End Task

Option C - Restart Computer
  (Stops all background processes)

Problem: Still getting ERR_CONNECTION_REFUSED
  Solution 1: Clear browser cache (Ctrl+Shift+Delete)
  Solution 2: Try incognito/private window
  Solution 3: Try http://127.0.0.1:8000 instead
  Solution 4: Check if server is running: netstat -ano | findstr :8000

Problem: Page loads but looks broken (no styling)
  Solution: Run npm install && npm run dev in another terminal

Problem: "419 Page Expired" error
  Solution: Clear cookies and try again
  
Problem: Can't login
  Solution: Check database connection in .env file
  
Problem: Port 8000 already in use
  Solution: Use different port: php artisan serve --port=8001

To check if server is running, open PowerShell and run:

  netstat -ano | findstr :8000 | findstr LISTENING

If you see output like:
  TCP    127.0.0.1:8000    0.0.0.0:0    LISTENING    [PID]
  
Then server is running! ✅

If you see nothing, server is not running ❌
  → Start it with: php artisan serve
