================================================================================
                    GENERATING APPLICATION KEY
================================================================================

The APP_KEY is crucial for encryption and security.

HOW TO GENERATE:
----------------
1. Open terminal in your project folder
2. Run: php artisan key:generate
3. This will automatically update your .env file
4. You should see: "Application key set successfully"

The key looks like: base64:abcdefghijklmnopqrstuvwxyz1234567890=

MANUAL METHOD (if artisan doesn't work):
-----------------------------------------
1. Visit: https://generate-random.org/laravel-key-generator
2. Click "Generate Laravel Key"
3. Copy the generated key
4. Paste in .env as: APP_KEY=base64:xxxxxxxxxxxx
